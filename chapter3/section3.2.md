## 核心流程


* 创建 .podspec 文件  ``pod spec create PublicLib``

![](Resource/3_2_3.png)

* .podspec 文件 校验

```
pod lib lint  校验本地podspec 文件有效性

pod spec lint 校验远程podspec 文件有效性

```


* pod trunk    注册 



```
pod trunk register '邮箱' '用户名' --description='描述'

```

向trunk服务器查询到自己的注册信息


```
pod trunk me

```


* 提交至远程索引库

```
pod trunk push xxxxx.podsepc --verbose 

```
